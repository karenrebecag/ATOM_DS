# @atomchat.io/mcp

MCP server that gives AI context about ATOM Design System components and tokens across React, Vue, Angular, and Astro.

## Usage

```
npx @atomchat.io/mcp
```

## Tools

- `atom_list_components` — Browse components
- `atom_get_component` — Full component docs
- `atom_get_tokens` — Design tokens
- `atom_search` — Search all
